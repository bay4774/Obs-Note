# Blender
## 视图
3D视图窗口：栅格平面，坐标轴，游标，导航器
**3D视图操作**：旋转【Alt+左键】，缩放【滚轮】，平移【中键】
三视图：前视图【-Y】，右视图【+X】，俯视图【+Z】
## 物体模式
> n键提供物体的相关参数侧边栏

可通过点击左侧边栏的三个选项分别实现下述三种处理，Ctrl可以控制旋转角度的整倍数，Shift可以控制缩放的精细程度。
移动（grab）【g, x/y/z】
旋转（rotate）【r, x/y/z】
缩放（scale）【s, x/y/z】
Shift +d 复制一个，也可以使用x/y/z控制新复制物体的移动方向
重命名 【F2】
删除 【x】
### 坐标
通过Shit + 鼠标右键重新定义游标位置，精确位置在编辑模式下选择后吸附游标
- 出生点
- 吸附定位
- 视图轴心
### 原点
默认为物体的几何中心
### 模型
多边形建模：通过操作点、边、面来改变形状
特点：由一小块一小块的面拼接而成，通常为三角形，四边形【模型里边时中空的】
三种基本元素
**选择快捷键**：Shift加选，Ctrl减选，A全选
可以使用左边侧栏倒数第二个键在新建物体【Shift控制正交】
- 点（vertex）
- 边（edge）
- 面（Face）

## 面操作
### 细分面
4种操作手段：菜单，工具，右键，快捷键
- 细分操作：选择面，右键细分，微调操作
- 删除操作：仅面，面和边【会影响相邻的面】
**面的细分，本质是对边的细分，一般针对四边形**
### 挤出面
左侧边栏可选择挤出面的操作，二级操作同时对法向操作
- 向外向内挤出
- 多个面挤出
- 法向，轴向挤出
- **快捷键 E** 挤出【Extrude】
### 内插面
左侧边栏选择，内插面相当于新建一个缩小的面，微调模式可以选择各面
- 鼠标放在圆圈处，鼠标按下，往里拖拽
- **快捷键 I** 内插 【Inset】
### 坚分面
- 将面分为多个三角形
- 融合面：将坚分面融合，按X选择融合
### 面的切割
使用刀来切割，会自动吸附顶点或边
- **快捷键K**启动【Knife】
- 使用Enter/Space确认完成，ESC/RMB取消。
- Shift会吸附中点
- Ctrl会忽略吸附
- A角度约束
- C 切穿
### 面的法向
Normal，垂直于面的方向
- 法向的显示：视图叠加层
- 反转法向：网格，法向，反转
- 检查面的方向：视图叠加层，面朝向，红色表示背面
- **Shift + N**：重新计算外侧，自动计算法相
### 面的填充
选中一圈顶点，按下**F键**得到面
- 栅格填充：面菜单下会有栅格填充，使用四边形填充，可以微调偏移量【**要求偶数边**】
### 透视模型
X-ray，右上角开启
- 方便选择后面看不见的顶点，边，面
- **Alt +  Z 快捷键**开启透视模式

## 边操作
删除：x
融并边：Ctrl + x
细分
滑移：g g，连按两次
### 边的挤出
轴向挤出 E x/y/z
### 边线倒角
可修改宽度和段数
倒角Bevel：**快捷键Ctrl + B**，滚轮调整段数
### 环切
Ring Cut：**快捷键Ctrl + R**，LMB控制切割，切割并滑移，滚轮调整段数
### 循环边与并排边
- 循环边：首尾相连的边，形成一圈 Loop
	- Shift + Alt + LMB
- 并排边，平行的边
	- 【Shift】+ Ctrl + Alt + LMB
- 可滑移  g g 
- 可容并，可倒角【多加几道】
- 法相收缩在左侧工具栏
- 卡线：加一道循环边，限制收缩范围
### 四边面优势
方便施加算法：循环边，并排边
### 桥接循环边
在两组循环边之间搭座桥
边数需要相同，来自同一物体，桥接后，检查面朝向
可应用于对接，打洞
